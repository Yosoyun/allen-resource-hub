# Backup snapshots

Point-in-time copies of the ALLEN Resource Hub, kept inside the project as a safety net.

- `index-2026-06-16.html` — full standalone copy of the site as of 2026-06-16. Open it directly in any browser.
- `allen-resource-hub-2026-06-16.zip` — zipped deployable bundle (index.html + README + vercel.json).

The live source of truth is the top-level `index.html`. These snapshots are backups only.
Restore by copying a snapshot back over `../index.html`.
